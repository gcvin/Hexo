### 向下取整
``` js
~~3.4          //=> 3
3.4|0          //=> 3
-3.4^0         //=> -3
-3.4 >> 0      //=> -3
```

### indexOf判断
``` js
var arr = [1,2,3];
if(!~arr.indexOf(4)){
  arr.push(4);
}
!!~arr.indexOf(1) //=> true
!!~arr.indexOf(5) //=> false
```

### 判断是否有重复的字母
function isIsogram(str){

}
